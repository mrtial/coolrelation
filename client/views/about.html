THE PROJECT

D3 is an incredibly power data visualization tool written in javascript. But in order to use it, it requires some basic understanding of HTML, SVG , CSS and Javascript.

Coolrelation is a D3 visualization generator for correlation data. It brings the powerful D3 to user who wants to use D3 to visualize their data but doesn't equippe with those prerequisite knowledge to use it. The re-useable D3 generator allows user to customize graph settings in order to get insights from their data.

It not only just pushes data into D3 model, but also uses python algorithm to calculate the optimal initial parameters so that user can extract meaningful relational structure out the graph itself.
